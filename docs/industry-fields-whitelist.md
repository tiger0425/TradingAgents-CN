# 行业分析字段白名单

> 定义各行业分析时必须包含（必含字段）和可选关注（可选字段）的指标字段。
> 该白名单由 `build_instrument_context()` 动态注入到 Agent 系统提示词中，
> 与 `industry_frameworks.json` 的反模式规则共同构成完整的行业分析约束。
>
> 新增行业时只需在此文件追加 `## {industry_name}` 章节即可。

---

## automotive

### 必含字段
- 月度销量
- 产能利用率
- 毛利率
- 新能源渗透率
- 市场份额
- 原材料成本占比
- 单车收入
- 单车利润

### 可选字段
- ROE
- 经营活动现金流/净利润比率
- 存货周转率
- 研发费用率
- 应收账款周转率
- PB/PE 估值倍数
- EV/EBITDA

## comm_cable

### 必含字段
- G.652.D散纤现货价（元/芯公里）
- 运营商年度光缆集采价格及招标量（亿芯公里）
- 光棒（预制棒）产能利用率
- 光棒自给率
- 铜/铝原材料成本占比
- 分业务毛利率拆分（光纤光缆/电力电缆/通信工程/海缆）
- 运营商集采市场份额排名
- 经营活动现金流/净利润比率（健康线≥0.8）
- 海外业务收入占比
- 特种光纤（G.654.E/空芯光纤）收入占比
- 海缆及电力电缆在手订单覆盖月数
- 存货周转天数

### 可选字段
- ROE
- 研发费用率
- 光缆出口增速
- 5G基站建设进度
- 光纤到户渗透率
- 行业CR4集中度变化
