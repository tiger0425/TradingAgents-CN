❌ FAIL (Test 2e): .format() crash — critical runtime bug
   PORTFOLIO_PARSE_PROMPT contains JSON schema with single curly braces:
     {
       "action": "...",
       ...
     }
   Python's str.format() interprets these as format specifiers.
   When line 191 calls PORTFOLIO_PARSE_PROMPT.format(message=...),
   it raises: KeyError('\n  "action"')
   
   Fix: Escape JSON braces as {{ and }} in the prompt string.
