# 草案：阶段1 A股助手开发

## 已确认决策

### 功能范围
- **IN（全部6项一起做）**：
  1. 实时行情 — akshare 新浪实时行情接口
  2. 条件单/预警 — 价格到达自动提醒（实时监控+推送）
  3. 短线异动监控 — 涨停/跌停/炸板/天地板
  4. 公告快读 — 个股最新公告抓取+LLM摘要
  5. 分析师研报 — 东方财富/同花顺研报抓取+摘要
  6. 持仓风险监控 — 大盘跌时自动评估持仓暴露

- **OUT**：券商交易执行、常驻daemon进程

### 技术决策
- 实时行情：akshare 新浪实时行情接口
- 监控方式：轮询命令（crontab 定时执行），不做常驻 daemon
- 通知推送：复用现有飞书/ServerChan/PushPlus 通道
- LLM摘要：复用现有 quick_think_llm + LLM链路
- 测试策略：**TDD**（RED → GREEN → REFACTOR）

### 运行方式
- 预警/异动监控：CLI 轮询命令，用户自行 cron 调度
- 不支持常驻 daemon 进程
